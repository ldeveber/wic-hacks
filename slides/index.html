What is Bootstrap?

Bootstrap is a front end framework built by Twitter for developing mobile responsive projects. Built on:

  • JavaScript
  • CSS
  • less

http://getbootstrap.com/

There is a sass port of Bootstrap available.

Customization

Theming and customization are very easy with Bootstrap. You can create your own version with only the plugins you need, as well as theming it to match colors and styles.

http://getbootstrap.com/customize/ has more information on customizing it for your own use.

Bootstrap and Ruby

Bootstrap can easily be included into any ruby project with the bootstrap-sass gem. Run: 


gem install bootstrap-sass
or, add the following to your gemfile: 

gem 'bootstrap-sass', '~> 3.1.1.1'

This is only one way to do it. You could of course just include it from the CDN.
